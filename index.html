Application Idea

In order to demonstrate both what I have learned in this class already as well as the ability to take those skills and build upon them with independent learning of my own, I would like to explore web development. My final project proposal is to construct a final project web page which expands upon the material reviewed in Sprints 2 and 3. I will make 15+ significant enhancements from what I accomplished in the Getting to Know You activities. One such enhancement I would like to make is to encorporate custom SVG art which I will create in Adobe Illustrator and import into the website. These logos and vector art files will be imported in areas that intuitively make sense for the site (such as a mail logo above a custom email hyperlink as seen above). The name of this application will be exploration-in-web-development.

From the user's perspective, the website will be aesthetically pleasing and intutive to use. SVG files will draw the eye where it's supposed to go and help to distinguish content on the page, though I have not yet decided what that might consist of. One idea I had is to break down the different topics covered in class and use the website as a sort of summation of the material covered in this course. For the end-user, the website should look as professional as I can achieve using my current capabilities (plus additional learning).

This project will be completed by me alone.

Application Features



Features which demonstrate independent learning are colored in dark magenta. Those that demonstrate understanding of topics covered in class are dark blue.






Prioritized:

  • 1. "Exploration in Web Development" website is hosted on Azure

  • 2. The website meets HTML and CSS coding standards

  • 3. The web page is accessible via FireFox and Chrome browsers

  • 4. 15+ significant enhancements are made from the skills invoked in "Getting to Know You"

  • 5. Implementation of custom SVG files imported from Adobe Illustrator to the website

Non-Prioritized:

  • The website integrates parallax scrolling

  • Custom animations are exported from Adobe After Effects and applied to elements of the web page

  • The website utilizes CSS animations

  • The web page includes interactive features which respond to what the user does
